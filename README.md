# How to use

## Import

```
import LoadingCat from 'react-native-loading-cat';
```

## Render

```
<View>
    <LoadingCat />
</View>
```

## Variety


Original Cat

```
<View>
    <LoadingCat variety="original" />
</View>
```

Black Cat

```
<View>
    <LoadingCat variety="black" />
</View>
```

White Cat

```
<View>
    <LoadingCat variety="white" />
</View>
```
